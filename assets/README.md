# Assets Directory

This directory contains visual assets for the DevWrapped 2025 project.

## Images

### devwrapped-hero.png
- **Purpose**: Hero image for README.md
- **Description**: Screenshot of the DevWrapped 2025 landing page showing the "Celebrate Your Code Journey" interface
- **Dimensions**: Recommended 1920x1080 or similar high resolution
- **Format**: PNG with transparency support

## Usage

Images in this directory are referenced in:
- README.md (hero image)
- Documentation files
- GitHub repository presentation

## Adding Images

To add the hero image:
1. Save your screenshot as `devwrapped-hero.png` in this directory
2. Ensure the image is high quality (preferably 1920x1080 or higher)
3. The README.md is already configured to display this image

## Image Guidelines

- Use PNG format for screenshots with UI elements
- Optimize file size while maintaining quality
- Use descriptive filenames
- Include alt text in markdown references
