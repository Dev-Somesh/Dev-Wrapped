# Changelog

All notable changes to DevWrapped 2025 will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- Initial release of DevWrapped 2025
- AI-powered GitHub analysis using Google Gemini
- Comprehensive developer insights and narratives
- Interactive ShareCard component with key statistics
- Detailed DevelopmentDossier with repository analysis
- Tech stack visualization with color-coded badges
- Activity grid showing contribution patterns
- Professional profile section with bio, company, location
- High-quality image export functionality
- Streamlined user experience (direct flow from input to results)
- Privacy-first architecture with zero data retention
- Serverless backend using Netlify Functions
- Responsive design with GitHub-inspired dark theme
- Animated background elements
- Comprehensive error handling and rate limiting

### Features
- **GitHub Integration**: Complete profile and repository analysis
- **AI Insights**: Personalized developer narratives and archetypes
- **Visual Components**: Beautiful cards and detailed dossiers
- **Export Capabilities**: Share-ready image generation
- **Security**: Server-side API key storage
- **Performance**: Optimized for fast loading and smooth interactions

### Technical Stack
- React 19 with TypeScript
- Tailwind CSS for styling
- Google Gemini AI for insights
- Netlify Functions for serverless backend
- Vite for build tooling
- html-to-image for export functionality

### Security & Privacy
- Zero data retention policy
- Client-side data processing
- Secure API key management
- No tracking or analytics cookies
- GDPR compliant architecture
