# Adding the Hero Image

## Quick Steps:

1. **Save the screenshot** you provided as `devwrapped-hero.png` in the `assets/` folder
2. **Commit and push** the changes:
   ```bash
   git add assets/devwrapped-hero.png
   git commit -m "feat: add hero image to README"
   git push origin main
   ```
